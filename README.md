# Calculator Web App

A simple and user-friendly Calculator application developed using HTML, CSS, and JavaScript. This project performs basic arithmetic calculations and provides an interactive interface for users.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Clear Display
* Responsive User Interface

## Technologies Used

* HTML
* CSS
* JavaScript

## Files

* calculator.html

## How to Run

1. Download or clone the project.
2. Open `calculator.html` in any web browser.
3. Start performing calculations using the calculator interface.

## Project Outcome

This project helped in understanding the fundamentals of web development, including HTML structure, CSS styling, and JavaScript functionality. It demonstrates how user interactions can be handled to perform real-time calculations.
